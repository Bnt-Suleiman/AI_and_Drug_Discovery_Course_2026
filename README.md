## Description of Data Curation Workflow
This repository contains a Jupyter Notebook demonstrating the retrieval and curation of bioactivity data for human EGFR from the ChEMBL database. The workflow includes filtering assays by confidence score, selecting standardized IC50 values, unit normalization, removal of duplicates, and preparation of a clean dataset suitable for QSAR modeling using Python.
